# Instalação
- Para clonar repositório em sua maquina. 
 ```git
git clone https://github.com/Blacklist14/Socket-chat 
```
- Instala dependências
```npm
npm i
``` 
 - Após instalação inicie o projeto no https://localhost:8080
```npm
npm start
```
